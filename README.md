# 霍普金森杆（Hopkinson Bar）数据处理软件

## 1. 霍普金森杆简介

###  工作原理
霍普金森杆（Hopkinson Bar）是一种用于测试材料在高应变率下力学性能的实验设备。该技术由英国科学家Bertram Hopkinson在20世纪初发明，主要用于研究材料在动态载荷下的应力-应变关系。

霍普金森杆系统主要由入射杆、透射杆和吸收杆组成，通过应力波在杆中的传播来测量材料的动态力学性能。

## 3. 参数说明

### 3.1 样品参数
- **样品编号**：唯一标识样品的编号
- **样品名称**：样品的名称或描述
- **样品材质**：样品的材料类型
- **截面直径**：样品的截面直径（mm）
- **样品长度**：样品的长度（mm）

### 3.2 杆参数
- **杆模式**：压杆或拉杆模式
- **杆材质**：杆的材料类型
- **截面直径**：杆的截面直径（mm）
- **Young's模量**：杆的弹性模量（GPa）
- **声速**：应力波在杆中的传播速度（m/s）
- **桥类型**：应变片桥路类型（1/4桥、半桥、全桥）
- **Gage因子**：应变片的灵敏系数
- **激励电压**：应变片桥路的激励电压（V）
- **标定系数**：系统标定系数
- **泊松比**：杆材料的泊松比
- **阻尼系数**：系统阻尼系数

### 3.3 计算参数
- **低频截止**：滤波器低频截止频率（Hz）
- **高频截止**：滤波器高频截止频率（Hz）
- **计算方法**：
  - 双波法（入射+透射）
  - 双波法（反射+透射）
  - 三波法
